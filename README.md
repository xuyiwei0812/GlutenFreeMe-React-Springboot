# Gluten Free Me
A healthy life website for gluten free recipes. 

by Luna Xu

Front end: React.js (Javascript + html + css, frontend framework)

Back end: Springboot (Java backend framework) + MyBatis (support for custom SQL)

Run front end: 

npm run install

npm start
